# DevOps Capstone Project

Customer Accounts Microservice built using Python Flask.

## Features
- Create an Account
- Read an Account
- Update an Account
- Delete an Account
- List all Accounts

## Technology Used
- Python
- Flask
- Docker
- Kubernetes
- GitHub Actions
